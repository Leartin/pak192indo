=====================
How To USE!!
=====================

A. Bahan 
1. File *.dat dan *.png yang akan dijadikan object / addons!
2. Niat
3. Keseriusan membaca tutorial / video
4. makeobj.exe & rar.exe
5. Compiler!

B. Step-by-Step
1. Taruh file *.dat dan *.png yang akan dijadikan object / addons! di folder ".\master\*" menurut kategorinya!
2. Start aplikasi "pak192_compiler.bat"
3. Tunggu proses compile.
4. Jika sudah, lihat hasil compile di folder "release"
5. DONE! Simple ? Kalo rumit berarti gabisa baca!

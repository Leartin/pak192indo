Cara Install : 
1. Buka Arsip *.rar
2. Extract Folder nya di 
 < root/pak192.indobeta/scenario/ > Atau
 < root/pak192.comic/scenario/ >
3. Buka simutrans ( Versi 2.1.3+ Only )
 < Download Executable at www.simutrans.com/download >
4. Load pakset 192.comic / pak192.indobeta
5. Klik Buka scenario..
6. Pilih scenarionya.. 
7. DONE

-

COMPABILITY : 
OS : All OS Support Simutrans 1.2.3+ 
RAM : 1 GB or Higher
Pakset : pak192.comic 1.4.1 and Pak192.indobeta r321
Simutrans : 1.2.3+ ( Dont Use Nightly )
